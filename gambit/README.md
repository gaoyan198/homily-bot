# GAMBIT

A 4–8 week swing-trading sleeve, run as a business: beat QQQ buy-and-hold
on the same dollars, skim the excess quarterly to fund long-term DCA.

Sibling project to [homily-bot](../homily-bot) — separate repo on purpose
(different safety contract: this one proposes sells; homily never does).
All methodology — pre-registered backtests, luck benchmarks, append-only
ledgers, validate-gated sends — is inherited from homily's hard lessons.
